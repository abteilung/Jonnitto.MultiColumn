MultiColumn System for [Neos](https://neos.io). Please bring your own css

Grid
----

Possible grid classes are:

**xs**  
`3, 4, 6, 8, 12`

**sm / md / lg / xl**  
`2, 3, 4, 6, 8, 9, 10`

Classes get rendered like this: `col-BREAKPOINT-NUMBER`, e. g. `col-sm-8`
